# Threat Model

> Status 2026-07-09: unchanged and still accurate — all five in-scope classes are
> covered by the 11 frozen rules + semantic confirmer and measured on the labeled
> corpus ([METRICS_AND_OUTCOMES.md](./METRICS_AND_OUTCOMES.md)).

## In Scope
- malicious tool descriptions
- hidden instructions
- risky permission combinations
- exfiltration-oriented text patterns
- task drift from poisoned instructions

## Out of Scope
- sandbox escapes
- OS-level exploits
- full browser agent hardening
